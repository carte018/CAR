Welcome to Amber!

Welcome to Amber -- the fictitious world of the CAR demo environment!
This demo environment comprises a complete and fully functioning CAR instance providing attribute release decisions to a Shibboleth IDP that's bilaterally federated with a set of four additional relying parties.

Set up your test accounts

If you're new here, you might want to start by creating a set of test users for yourself. As you use the various test services in the Sliced Bread environment, you'll necessarily make persistent changes to the privacy settings of the users you log in with. To avoid confusion when multiple people are interacting with the environment, you should set up a collection of personal test users for yourself. It's quick and easy -- just visit:
  • Test Account Creation
and supply a first and middle initial and a last name -- we'll take care of the rest!
